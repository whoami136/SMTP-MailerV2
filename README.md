# 📧 SMTP Mailer V2 (Bird Edition)

SMTP Mailer V2 is an advanced Python SMTP email sender tool with a stylish bird-themed CLI interface and HTML email support.

---

## 🐦 Features

- Bird-themed colorful terminal UI
- ⚠️ HTML “yellow warning” email design
- 📩 SMTP email sending (Gmail supported)
- 📎 File attachment support
- 🔥 High priority option (Yes / No)
- 🎨 Clean structured email formatting
- 🧠 Simple interactive workflow
