# 📧 SMTP Mailer V2 (Bird Edition)

SMTP Mailer V2 is an advanced Python-based email sending tool that uses SMTP servers to send structured HTML emails with attachments, priority settings, and a visually enhanced CLI interface.

---

## 🚀 Features

- 🐦 Bird-themed colorful CLI interface
- ⚠️ HTML email warning banner (yellow “Important message” style)
- 📩 Send emails via SMTP (Gmail supported)
- 📎 File attachment support
- 🔥 High priority email option (Yes/No toggle)
- 🎨 Clean formatted email templates (HTML + text fallback)
- 🧠 Simple interactive terminal UI

---

## 📦 Requirements

- Python 3.x
- SMTP access (Gmail App Password recommended)
